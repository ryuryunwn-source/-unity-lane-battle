# マイモン：侵攻ライン (Lane Invasion)

カードゲームに「レーン制ストラテジー（タワーディフェンス）」のメカニクスを移植した2人対戦ゲーム。
TCG版（別リポジトリ unity-ai-tcg-online）のネットワーク基盤・カードデータ・アート資産を流用して新規構築。

## ルール
- 盤面は **5レーン × 5セル**。Player1は左端→右へ、Player2は右端→左へ進軍。
- 自分のターン: MP獲得 → ドロー → 自陣の端列セルにモンスターを召喚（コスト消費）→ ターン終了。
- ターン終了時、自軍ユニットが敵方向へ1セル前進。
  - 進む先に敵ユニット → 自動戦闘（互いにATK分ダメージ、HP0で破壊）。
  - 敵陣を突破して相手ベースに到達 → 相手ベースHPにATK分ダメージ。
- 相手のベースHPを0にしたら勝利。
- 山札切れ後は疲労ダメージでベースHPが削れる（膠着防止）。

## セットアップ（Unity Editor）
1. プロジェクトを Unity 6000.4.4f1 で開く
2. メニュー **Tools → Lane → Build Scene** を実行（5×5盤面・UI・マネージャを生成）
3. シーンを保存して **Play**
4. タイトルで「1台で対戦」を選ぶとローカル2人対戦が開始

## 操作
- 手札のカードをクリックで選択 → 自陣端列の空きセルをクリックで召喚
- 「ターン終了」ボタンで進軍フェーズ実行＆相手ターンへ

## 主要スクリプト (Assets/Scripts/Lane/)
- `LaneGameManager` … 進行・進軍・戦闘・勝敗
- `LanePlayer` … ベースHP/MP/デッキ/手札
- `LaneUnit` … レーン上のモンスター
- `LaneBoard` … 5×5占有モデル
- `LaneUI` / `LaneCell` / `LaneHandCard` … 表示と入力
- `LaneSetup` … ランタイム初期化
- `LaneDeckFactory` … サンプルデッキ生成
- `Editor/LaneSceneBuilder` … シーン自動構築
